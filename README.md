# OSCE — Edema Agudo Pulmonar Perfil B

Protocolo OSCE estruturado para Edema Agudo Pulmonar Perfil B, desenvolvido como material de memorização médica aplicada à simulação realística.

## Sobre o material

- Algoritmo ACLS-like com sequência pedagógica fragmentada
- Flashcards interativos por fase
- Perguntas e respostas (accordion)
- Mnemônico APTM — Aórtico, Pulmonar, Tricúspide, Mitral
- Barra de progresso lateral com tracking por fase
- Fluxograma visual completo ao final
- Baseado nas **Diretrizes SBC 2023–2024**
- Direcionado ao **TEC 2026**

## Conteúdo

| Fase | Tema | Tempo |
|------|------|-------|
| F0 | Chegada e impressão inicial | T = 0 min |
| F1 | O₂ + FiO₂ + Monitor | T = 0–3 min |
| F2 | ABCD fragmentado | T = 3–8 min |
| F3 | Exames laboratoriais | T = 5–8 min |
| F4 | Anamnese com acompanhante | T = 3–5 min |
| F5 | Sonda vesical + Furosemida | T = 8–10 min |
| F6 | Nitrato EV — Tridil vs Nipride | T = 10–12 min |
| F7 | Bloqueio sequencial do néfron | T = 30–40 min |
| F8 | Reconciliação de medicamentos | T = 40–50 min |
| F9 | Encerramento e orientação | T = 60 min |
